Comment écrire un paragraphe en HTML ?

La balise HTML <p>

La balise <p> permet d'écrire des paragraphes au sein d'un code HTML dans la partie <body>.

Cette balise est de type block et occupera donc toute la largeur disponible de l'écran.

Pour rédiger un paragraphe, vous devez l'insérer entre une balise ouvrante et fermante.

Exemple

<p>Je suis un paragraphe</p>
